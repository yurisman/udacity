difcil medir a qualidade de um modelo dado sem quantificar seu desempenho durante o treinamento e teste. Para este projeto, voc ir calcular o coeficiente de determinao, R2, para quantificar o desempenho do seu modelo. O coeficiente de determinao uma estatstica til no campo de anlise de regresso uma vez que descreve o quo "bom" a capacidade do modelo em fazer estimativas.

Os valores para R2 tm um alcance de 0 a 1, que captura a porcentagem da correlao ao quadrado entre a estimativa e o valor atual da varivel alvo. Um modelo R2 de valor 0 sempre falha ao estimar a varivel alvo, enquanto que um modelo R2 de valor 1, estima perfeitamente a varivel alvo. Qualquer valor entre 0 e 1 indica qual a porcentagem da varivel alvo (ao utilizar o modelo) que pode ser explicada pelos atributos. Um modelo pode dar tambm um R2 negativo, que indica que o modelo no melhor do que aquele que estima ingenuamente a mdia da varivel alvo.

Questão 8 - Validação Cruzada¶

  • Quais benefcios essa tcnica proporciona para busca em matriz ao otimizar um modelo?

Dica: Lembre-se de expllicar o que significa o 'k' da validao-cruzada k-fold, como a base de dados dividida e quantas vezes ela executada.

Resposta: Validao consiste em separar uma parte dos dados para ser utilizada em decises relacionadas aos hiperparmetros do modelo. Como consequncia, uma quantidade menor de dados est disponvel para o treinamento. Com o intuito de resolver esse problema e ao mesmo tempo realizar a validao cruzada, desenvolveu-se o mtodo validao-cruzada k-fold. O mtodo consiste em: Dados so divididos em k grupos (por esse motivo o nome k-fold), geralmente os dados so selecionados de forma aleatria para eliminar qualquer vis relacionado a ordem dos dados. Modelo treinando k vezes, sendo que em cada interao, 1 grupo utilizado para validao. modelo final a mdia dos modelos e o score a mdia dos scores de cara iterao. Protanto, temos como benefcios: modelo utiliza todos os pontos selecionados para treinamento e ao mesmo tempo, pode-se determinar os scores de validao (utilizados para a escolha dos hiperparmetros).
